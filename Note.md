# 환경설정
1. 프로젝트 폴더 생성
2. express-generator 를 이용하여 기본 구조 생성
3. npm install --save-dev  
>nodemon  
서버 on/off 하지 않고 변경사항을 바로 확인하는데 도움을 줌

4. 아래 패키지 설치
    - multer
    - multer-s3
    - mongoose (DB 관리)
    - socket.io (실시간채팅)
    - jsonwebtoken (회원가입 기능)
    - bcrypt (회원가입, 로그인)
    - aws-sdk (aws 접근)
